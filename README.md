# 💕 두근두근 엄마연구 시뮬레이터

나만의 "엄마"를 직접 만들고, 일주일간 함께 일상을 보내며 엄마를 알아가는 시뮬레이션 게임입니다.

엄마의 얼굴과 성격을 커스터마이징하고, 3D 방에서 엄마를 관찰하고, 외출·일상 이벤트를 함께 겪으며, 일주일 뒤 엄마와의 추억으로 엔딩을 맞이합니다.

---

## ▶️ 실행 방법

**`index.html` 파일을 더블 클릭해서 실행하세요.**

그게 전부입니다. 별도의 설치나 프로그램은 필요 없으며, 인터넷 브라우저(크롬, 사파리, 엣지 등)에서 바로 열립니다.

> ⚠️ **`index.html` 외의 다른 파일은 절대 건드리지 마세요.**
> `js`, `css`, `assets` 폴더와 그 안의 파일들은 게임이 작동하는 데 꼭 필요한 파일입니다.
> 옮기거나, 이름을 바꾸거나, 삭제하면 게임이 정상적으로 실행되지 않습니다.
> 폴더 전체(이 README가 들어있는 폴더)를 통째로 보관해 주세요.

---

## 🎮 게임 흐름

1. **엄마 만들기** — 얼굴(얼굴형·눈·코·입·머리·의상·액세서리)과 성격(MBTI 4문항)을 골라 나만의 엄마를 완성합니다.
2. **일상 관찰** — 3D 방에서 엄마가 지내는 모습을 지켜봅니다.
3. **함께하는 이벤트** — 식당, 마트, 영화관, 공원, 거리 등으로 외출하거나 집에서 일상을 보내며 다양한 선택을 합니다. 선택에 따라 엄마와의 애정과 성격 수치가 달라집니다.
4. **엔딩** — 일주일이 지나면, 그동안 쌓인 추억과 애정에 따라 달라지는 엔딩을 보게 됩니다.
5. **온라인 통계** — 다른 플레이어들이 만든 엄마의 성별·성격 분포, 이벤트 선택 통계를 함께 볼 수 있습니다.

## 🛠️ 기술 정보 (참고용)

- **Three.js r128** — 3D 방·캐릭터 렌더링
- **Canvas 2D API** — 얼굴 파츠 드로잉 & 텍스처 생성
- **Firebase Firestore** — 온라인 통계 집계
- **localStorage** — 캐릭터·진행 상황 저장

> 인터넷 연결은 온라인 통계 기능에만 필요하며, 게임 진행 자체는 오프라인에서도 가능합니다.
